## 1. 로그인 페이지 (`LoginPage`)

### 레이아웃
- **배경**: `#F7F8FA` (연한 회색)
- **중앙 카드**: 최소 380px, 최대 440px, `border-radius: 24px`, 그림자 효과
- **폰트**: Noto Sans KR

### 흐름 (위에서 아래로)
```
┌─────────────────────────────────┐
│         키움 트레이딩              │  ← 22pt Bold, 중앙 정렬
│   안전하고 편리한 해외주식 자동매매   │  ← 11pt, #8B95A1
│                                 │
│  서버 선택 (실전 / 모의)           │  ← 라벨 13px, #4E5968
│  ┌─────────────────────────┐    │
│  │ 실전 서버 (api.kiwoom..) ▼│    │  ← ComboBox, 48px height
│  └─────────────────────────┘    │
│                                 │
│  앱키                            │
│  ┌─────────────────────────┐    │
│  │ 발급받은 앱키를 입력하세요   │    │  ← placeholder
│  └─────────────────────────┘    │
│                                 │
│  시크릿                          │
│  ┌─────────────────────────┐    │
│  │ ●●●●●●●●●●●●●●●        │    │  ← Password mode
│  └─────────────────────────┘    │
│                                 │
│  계좌번호                        │
│  ┌─────────────────────────┐    │
│  │ 예: 5747-1946 또는 57471946│   │
│  └─────────────────────────┘    │
│  숫자만 입력하세요 ( - 는 자동 제거)│  ← 힌트, 11px, #B0B8C1

│                                 │

│  │       연결하기              │    │  ← Primary #3182F6, 50px

│                                 │

└─────────────────────────────────┘

    © 2026 Alpha Predator                ← 푸터, 10px, #B0B8C1
```

### 상호작용
| 동작 | 결과 |
|------|------|
| Enter 키 (앱키 필드) | 시크릿 필드로 포커스 이동 |
| Enter 키 (시크릿 필드) | 계좌번호 필드로 포커스 이동 |
| Enter 키 (계좌번호 필드) | 로그인 시도 |
| "연결하기" 클릭 | 유효성 검사 → 로딩 상태 → 성공/실패 |
| 로딩 중 | 버튼 "⏳ 연결 중입니다…", 모든 입력 비활성화 |
| 연결 실패 | 빨간 에러 메시지, 입력 재활성화 |
| 연결 성공 | 녹색 성공 메시지 → TradingScreen으로 전환 |
| "게스트로 시작" | Guest 모드로 TradingScreen 전환 |
| "텔레그램 연결" | TelegramDialog 모달 열기 |

## 2. 텔레그램 설정 다이얼로그 (`TelegramDialog`)

### 레이아웃
- **크기**: 460 × 560px 고정
- **배경**: `#F7F8FA`, 내부 카드 `#FFFFFF`, `border-radius: 20px`

### 흐름
```
┌─────────────────────────────────┐
│  텔레그램 알림 설정                │  ← 18pt Bold
│  봇 토큰과 채팅 ID를 입력하면      │
│  매매 알림을 텔레그램으로 받을 수    │  ← 12px, #8B95A1
│  있습니다.                        │
│                                 │
│  봇 토큰                         │
│  ┌─────────────────────────┐    │
│  │ ●●●●●●●●●●●●●●●        │    │  ← Password
│  └─────────────────────────┘    │
│                                 │
│  채팅 ID                         │
│  ┌─────────────────────────┐    │
│  │ 봇과 대화한 채팅방 ID       │    │
│  └─────────────────────────┘    │
│                                 │
│  ☑ 텔레그램 알림 사용              │
│  ☑ 설정 저장                      │
│                                 │
│  ┌─────────────────────────┐    │
│  │     테스트 전송              │    │  ← Outline #3182F6
│  └─────────────────────────┘    │
│                                 │
│  ┌───────────────┐ ┌────────┐  │
│  │     저장        │ │  닫기   │  │  ← Primary / Secondary
│  └───────────────┘ └────────┘  │
│                                 │
│  ✅ 텔레그램 설정이 저장되었습니다.  │  ← 상태 메시지
└─────────────────────────────────┘
```

### 상호작용
| 동작 | 결과 |
|------|------|
| "테스트 전송" 클릭 | QThread로 비동기 API 호출, 로딩 표시 |
| 테스트 성공 | ✅ 녹색 상태 메시지 |
| 테스트 실패 | ❌ 빨간 에러 메시지 |
| "저장" 클릭 | 토큰 검증 → 저장 → 600ms 후 자동 닫기 |
| "닫기" 클릭 | 다이얼로그 닫기 (변경 없음) |
